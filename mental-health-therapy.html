Project Overview

This innovative VR-based Android application offers immersive therapy experiences for individuals suffering from ADHD, claustrophobia, and anxiety-related stress. Built using Unity (C#) and Firebase, and optimized for use with affordable VR boxes, it bridges the gap between traditional therapy and tech-based accessibility.

üí° Highlights & Innovations

Accessibility First

The application was built with a focus on making mental health therapy accessible to everyone:

  • Affordable mental health therapy for low-resource environments
  • Fully immersive therapy experience built for Android
  • Works with inexpensive VR headsets that use smartphones

Technical Innovation

The project leverages several technical innovations:

  • Modular design with extensibility for future mental health modules
  • Adaptive input with VR headset touch + gyro-based movement
  • Real-time analytics and session data capture via Firebase
  • Optimized 3D environments to run smoothly on mobile hardware

Clinical Impact

Initial feedback has shown promising results:

  • Users with ADHD reported better focus and less impulsivity
  • Claustrophobic users felt increased comfort with exposure control
  • Therapists noted ease of integration into guided therapy sessions
  • Feedback praised its affordability and accessibility for non-clinical users
